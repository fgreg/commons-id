This package contains factories and implementation classes to generate unique identifiers.

The implementation is divided into the following subpackages.

  • The main package contains uid interface definitions, abstract base classes for identifier generators, generator factories and an IdentifierUtils convenience class that maintains singleton generators and exposes static methods for generating identifiers of the various kinds.
  • The random subpackage contains implementations of generators that return identifiers that appear to be random.
  • The serial subpackage contains generators that yield sequences of identifiers that follow a regular sequence (e.g., increasing numeric values).
  • The uuid subpackage contains implementations of the IETF Draft UUID specification
